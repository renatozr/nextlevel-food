## Getting Started

### Prerequisites

- NPM
- AWS S3

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/renatozr/nextlevel-food.git
   ```
2. Enter the directory
   ```sh
   cd nextlevel-food
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Fill in the environment variables
   ```sh
   AWS_ACCESS_KEY_ID=
   AWS_SECRET_ACCESS_KEY=
   AWS_BUCKET_NAME=
   ```
5. Run the project
   ```sh
   npm run dev
   ```
